# Idea: Number Pattern Examples

✍ Summary:
- Additive: 12, 17, 22, 27, 32 → +5 each time
- Subtractive: 98, 92, 86, ... → −6 each time
- Multiplicative: 4, 12, 36, ... → ×3
- Mixed pattern: 5, 7, 11, 19, 35 → Adding doubling values
