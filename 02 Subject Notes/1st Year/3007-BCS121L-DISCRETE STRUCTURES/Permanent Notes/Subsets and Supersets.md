# Idea: Subsets and Supersets

✍ Summary:
- A ⊆ B if all elements of A are in B.
- A proper subset: A ⊂ B if A ⊆ B and A ≠ B.
- ∅ is a subset of all sets.
- Special sets: Universal set (U), Empty set (∅).
