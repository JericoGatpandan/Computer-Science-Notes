# Idea: Bit, Byte, and Data Storage Units

✍ Summary:
- **Bit**: Smallest unit of digital data (0 or 1).
- **Byte**: 8 bits.
- Storage units increase by a factor of 1,024:
  - **KB** = 1,024 bytes
  - **MB** = 1,024 KB
  - **GB** = 1,024 MB
  - **TB** = 1,024 GB
  - **PB**, **EB**, **ZB**, **YB** follow in the same pattern.
