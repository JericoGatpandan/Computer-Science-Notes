# Idea: FOPL Syntax and Semantics

✍ Summary:
- Valid sentences formed using:
  - Predicates
  - Logical operators: ~, ∧, ∨, →, ↔
  - Quantifiers: ∀, ∃
- Semantics:
  - ∀x P(x): TRUE only if P holds for every x in U
  - ∃x P(x): TRUE if P holds for some x in U
