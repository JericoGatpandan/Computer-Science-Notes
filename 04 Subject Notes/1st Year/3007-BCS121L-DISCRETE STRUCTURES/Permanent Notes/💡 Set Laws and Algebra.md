# Idea: Set Laws and Algebra

✍ Summary:
- **Identity Law**: A ∪ ∅ = A; A ∩ U = A
- **Domination Law**: A ∪ U = U; A ∩ ∅ = ∅
- **Idempotent Law**: A ∪ A = A
- **Double Complement**: (A')' = A
- **De Morgan’s Laws**:
  - (A ∪ B)' = A' ∩ B'
  - (A ∩ B)' = A' ∪ B'
