# Idea: What is a Set?

✍ Summary:
- A set is a **collection of unordered, distinct, and well-defined** objects (elements).
- Example: {1, 2, 3} = {3, 2, 1}; {1, 1, 2} = {1, 2}
- Introduced by Georg Cantor.
