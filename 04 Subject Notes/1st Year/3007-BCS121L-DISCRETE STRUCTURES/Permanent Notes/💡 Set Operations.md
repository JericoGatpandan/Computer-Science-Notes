# Idea: Set Operations

✍ Summary:
- **Union (A ∪ B)**: elements in A or B or both.
- **Intersection (A ∩ B)**: elements in both A and B.
- **Difference (A - B)**: elements in A not in B.
- **Complement (A')**: elements in U not in A.
- **Symmetric Difference (A △ B)**: (A ∪ B) - (A ∩ B)
