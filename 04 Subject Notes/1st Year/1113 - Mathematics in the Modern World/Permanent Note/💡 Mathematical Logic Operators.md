# Idea: Mathematical Logic Operators

✍ Summary:
- Logical Connectives:
  - **Negation (~s)** → “It is not the case that...”
  - **Conjunction (s ∧ t)** → “s and t”
  - **Disjunction (s ∨ t)** → “s or t”
  - **Implication (s → t)** → “If s, then t”
- Truth Tables guide the evaluation of each connective.
