# Idea: 💡 User Interfaces (CLI & GUI)

✍ Summary:
- - **Command-Line Interface (CLI)**: Text-based. Requires typed commands (e.g., MS-DOS).
    
- **Graphical User Interface (GUI)**: Visual. Uses icons and menus (e.g., Windows, macOS).
