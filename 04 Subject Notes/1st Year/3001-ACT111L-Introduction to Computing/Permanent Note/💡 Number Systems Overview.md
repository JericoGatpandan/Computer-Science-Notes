# Idea: Number Systems Overview

✍ Summary:
- **Binary** – Base 2 (e.g., 1011)
- **Decimal** – Base 10 (regular counting)
- **Octal** – Base 8
- **Hexadecimal** – Base 16

These systems are used in computing for representing and processing data efficiently.
