# Idea: Binary and Base Conversions

✍ Summary:
- **Binary → Octal**: Group binary digits in sets of 3
- **Binary → Hexadecimal**: Group binary digits in sets of 4
- **Octal → Decimal** and **Hexadecimal → Decimal** use place value multiplication
